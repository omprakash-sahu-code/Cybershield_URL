# Security Policy

## Supported Versions

Below are the versions of CyberShield currently receiving active security support and updates:

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within CyberShield, please do **NOT** open a public issue. Instead, follow these steps:

1. Send a detailed report to **security@cybershield.org** explaining the issue.
2. Include steps to reproduce the vulnerability, along with any relevant payloads or screenshots.
3. We will acknowledge receipt of your report within 48 hours and work with you to coordinate a security release.
